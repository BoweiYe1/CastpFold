CASTpFold Update

  • The database is enlarged, now containing both PDB and AlphaFold2 predicted structures.
    All 'non-fragment' sequences in Uniprot have representative AF2 structures, totaling 30 million.

  • Similar pockets are now provided for surface and protein-protein interacting pockets.
    The surface pocket database has 4.1 million members, and the protein-protein interaction (ppi) pocket database contains 500 thousand.

  • Predicted functional pockets and residues are provided for AF2 structures.
    The corresponding Gene Ontology (GO) terms/Enzyme Commission (EC) numbers will be provided.

Webpage Update

  • Changing the view in the molecular viewer is now more intuitive.

  • We remade the UI of the CASTp server website.
    Now you can find more helpful information via the website.

  • We fixed the bug that made the CASTp server website not compatible with the Safari browser.

  • We fixed the issues that caused the CASTp server sometimes to not be displayed properly on mobile devices.
